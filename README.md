# BBC中国航天报道爬虫

此项目爬取BBC网站上关于中国航天的英文报道，收集文章内容、图片和视频。

## 功能

- 爬取BBC网站上关于中国航天的英文报道（目标200篇以上）
- 提取并保存文章正文内容（.txt格式）
- 下载文章中的图片（.jpg、.png等格式）
- 下载文章中的视频
- 将内容分别存储在不同文件夹中

## 文件结构

- `articles/`: 存放文章正文（.txt文件）
- `images/`: 存放文章中的图片
- `videos/`: 存放文章中的视频
- `bbc_crawler.py`: 主爬虫脚本
- `requirements.txt`: 项目依赖

## 使用方法

1. 安装依赖：
```
pip install -r requirements.txt
```

2. 运行爬虫：
```
python bbc_crawler.py
```

## 注意事项

- 爬虫仅收集正文部分的图片和视频，不包括网站上的图标和广告等不相关内容
- 爬取的文章将以.txt格式保存，文件开头包含文章标题和网址，末尾列出图片和视频清单
